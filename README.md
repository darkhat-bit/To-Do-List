A straightforward, command-line To-Do List application built in Python. This program allows users to manage their tasks by adding, viewing, marking tasks as complete, and saving the list for later use.

## âœ¨ Features

* **Add Tasks:** Quickly add new items to your to-do list.
* **View List:** Display the entire list with status indicators (e.g., [ ] or [X]).
* **Mark Complete:** Change the status of a task to complete.
* **Delete Tasks:** Remove tasks from the list permanently.
* **Data Persistence:** Automatically saves your tasks to a local file (e.g., `todo.txt`) and loads them when the application starts.
